2. Requirements
-------------------------------------------------------------------------------

OS: 
Any OS having Java JDK 1.6+ installed and properly configured!
    We recommend using SUN JDK available at java.sun.com

Hardware: 
Decent CPU & RAM

Software:
Java JDK 1.6+
MySql 4.x (5.x recommended)

Note:
Dont play and host the server from the same computer. 
If you do so and get socket errors set the ram amount of the gameserver to
to 512Mb or less.
